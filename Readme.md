# Open Thermal Camera - KiCad PCB
This repository contains KiCad source files for Open Thermal Camera PCB.
Under releases you can find:
- Interactive BOM (.html)
- KiCost BOM (.xlsx)
- Gerbers
- Position files (.pos & .rpt)
